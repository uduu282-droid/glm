## 🔥 PRO TIPS

### 1. Rotate Models for Best Results:
```python
models_to_try = [
    "anthropic/claude-opus-4-6",
    "openai/gpt-5.2",
    "google/gemini-3-pro",
    "deepseek/deepseek-v3-2"
]

# Try different models for same prompt
for model in models_to_try:
    MODEL = model
    # send_message()...
```

### 2. Match Model to Task:
- **Coding:** Claude Opus or DeepSeek
- **Writing:** Claude Opus or GPT-5.2
- **Chat:** GPT-5.2 or Gemini
- **Speed:** Claude Haiku or GPT-5 nano
- **Long docs:** Kimi K2.5 or Claude Opus

### 3. Test Before Committing:
Use the Playground to test which model works best for your use case before integrating into your script!
