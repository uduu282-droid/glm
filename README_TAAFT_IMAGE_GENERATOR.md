### Security Mechanism

The API is protected by dynamically generated headers created by client-side JavaScript:

#### `se_accept_encoding` Generation Algorithm

Discovered from page source (line ~849):

```javascript
// Structure:
se_accept_encoding = [4 random chars] + [hostname] + [path with 17 random chars inserted] + [20 random chars]

// Example:
ABCD + theresanaiforthat.com + /api/generaXYZ...te/ + EFGHIJKLMNOPQRSTUVWXYZ
```

**Python Implementation**:
```python
import random
import string

def generate_token(hostname='theresanaiforthat.com', path='/api/generate/'):
    prefix = ''.join(random.choices(string.ascii_letters + string.digits, k=4))
    random_chars = ''.join(random.choices(string.ascii_letters + string.digits, k=17))
    suffix = ''.join(random.choices(string.ascii_letters + string.digits, k=20))
    
    # Insert random chars into path
    obfuscated_path = path[:4] + random_chars + path[4:]
    
    return prefix + hostname + obfuscated_path + suffix
```

## 🛠️ Implementation Files

### Working Solutions

#### 1. Puppeteer Browser Automation ⭐ BEST
**File**: `taafft_puppeteer.js`

```javascript
const puppeteer = require('puppeteer');

async function generateImage(prompt) {
    const browser = await puppeteer.launch({ headless: 'new' });
    const page = await browser.newPage();
    
    await page.goto('https://theresanaiforthat.com/@taaft/image-generator/');
    await page.type('textarea[placeholder*="Describe"]', prompt);
    await page.click('button:has-text("Generate")');
    
    // Wait for image
    await page.waitForFunction(() => {
        return document.querySelector('img[src*="generated"]');
    }, { timeout: 120000 });
    
    // Download image
    const imageData = await page.evaluate(() => {
        const img = document.querySelector('img[src*="generated"]');
        return img.src;
    });
    
    await browser.close();
    return imageData;
}
```

## 🐛 Troubleshooting

### Issue: "Generate button not found"
**Solution**: The page structure may have changed. Update selectors in `taafft_puppeteer.js`.

```javascript
// Try alternative selectors
await page.waitForSelector('button[type="submit"]');
// OR
await page.waitForSelector('.generate-button');
// OR
await page.waitForSelector('#generate-btn');
```

### Issue: Timeout during generation
**Solution**: Increase timeout or check network connection.

```javascript
await generateTaaftImage(prompt, {
    timeout: 180000  // 3 minutes
});
```

### Issue: Puppeteer won't launch
**Solution**: Install missing dependencies.

**Windows**:
```bash
npm install puppeteer
```

**Linux**:
```bash
sudo apt-get install -y \
    chromium-browser \
    libx11-xcb1 \
    libxcomposite1 \
    libxcursor1
```

### Issue: Images not saving
**Solution**: Check write permissions and disk space.

```bash
# Check current directory permissions
ls -la

# Specify absolute path
outputPath: '/tmp/my_image.png'
```
